Learning express-generator

安裝 express-generator scaffolding 工具

npm install -g express-generator

建立專案

express <專案名稱>

切換到專案資料夾安裝 npm 套件

cd <專案名稱> && npm install

開啟伺服器

./bin/www

當你登入 http://localhost:3000/ 會出現 log

開啟 debug log

DEBUG=<filter> ./bin/www

filter 的格式可以有 wildcard (*)

例如：

DEBUG=* ./bin/www
